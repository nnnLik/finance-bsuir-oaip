# Тест-сценарии: вкладки аналитики

## Подготовка

- Убедиться, что есть минимум 2 счета:
  - один с транзакциями;
  - один пустой.
- В счете с транзакциями должны быть доходы и расходы минимум в 2 категориях и 2 месяцах.

## Сценарии

### 1) Пустой счет

1. Выбрать пустой счет в основном окне.
2. Перейти в страницу `Аналитика`.
3. Проверить все вкладки: `Обзор`, `Расходы`, `Доходы`, `Тренды`.

Ожидание:
- На каждой вкладке отображается сообщение `Нет данных для выбранного счета/фильтра`.
- Нет «залипания» графиков/данных от предыдущего счета.

### 2) Переключатель охвата (Активный/Все)

1. На странице `Аналитика` установить scope: `Активный счет`.
2. Зафиксировать значения на вкладке `Обзор`.
3. Переключить scope на `Все счета`.
4. Повторно проверить `Обзор`, затем `Расходы`, `Доходы`, `Тренды`.

Ожидание:
- Все 4 вкладки обновляются в соответствии с выбранным scope.
- В режиме `Все счета` показатели не меньше, чем в `Активный счет` (при наличии данных на других счетах).

### 3) Корректность графиков на вкладке Обзор

1. Для выбранного scope сравнить:
   - доходы на графике,
   - расходы на графике,
   - KPI-значения сверху.
2. Сверить с данными таблицы операций.

Ожидание:
- Значения на столбцах соответствуют KPI.
- Баланс = доходы - расходы.
- Количество операций соответствует фактическому числу операций в scope.

### 4) Корректность вкладки Расходы

1. Открыть вкладку `Расходы`.
2. Сверить категории и суммы с данными операций (только расходы).

Ожидание:
- Категории расходов присутствуют корректно.
- Суммы по категориям соответствуют данным операций.
- Пустых/некорректных категорий нет (кроме `(без категории)` при реально пустых данных).

### 5) Корректность вкладки Доходы

1. Открыть вкладку `Доходы`.
2. Сверить категории и суммы с данными операций (только доходы).

Ожидание:
- Категории доходов и их суммы корректны.
- Значения соответствуют данным операций.

### 6) Корректность вкладки Тренды

1. Открыть вкладку `Тренды`.
2. Проверить подписи по месяцам и две линии (`Доходы`, `Расходы`).

Ожидание:
- Месяцы идут в хронологическом порядке.
- На каждом месяце значения линий соответствуют суммам за месяц.
- Показываются максимум последние 6 месяцев с операциями.

### 7) Обновление при смене счета

1. Находясь на странице `Аналитика`, переключать счета из верхнего селектора.
2. Не покидать страницу аналитики между переключениями.

Ожидание:
- Данные на активной вкладке и остальных вкладках обновляются сразу.
- Нет визуальных артефактов и смешивания данных счетов.
