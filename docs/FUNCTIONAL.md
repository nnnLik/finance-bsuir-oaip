# Функционал приложения «Учёт доходов и расходов»

## Назначение

Десктоп-приложение (Delphi VCL): ведение финансовых операций с хранением в SQLite, фильтрацией, аналитикой, поддержкой нескольких счетов и экраном настроек (CSV импорт/экспорт и управление категориями).

---

## Данные на диске

| Файл | Назначение |
|------|------------|
| `finance.db` (рядом с exe) | Основная SQLite база: счета, операции, категории, настройки. |
| `*.csv` | Экспорт/импорт операций через экран «Настройки». |

По умолчанию используется `finance.db` в каталоге приложения.

---

## Главное окно (`UnitMainShell`)

### Боковая панель (иконки)
- **Главная** — таблица операций активного счета.
- **Аналитика** — вкладки со сводками и графиками; область **активный счет** / **все счета**.
- **Настройки** — CSV, счета, категории.

### Верхняя полоса
- **Текущий баланс** — баланс активного счета.
- **Счет** — выбор активного счета.

### Страница «Главная» (`UnitMain`)
- **Фильтр по категории**.
- **Таблица**: дата, тип, категория, описание, сумма, иконки редактирования/удаления.
- **Добавить запись** — модальный диалог `UnitTransactionEditor`.

### Страница «Аналитика» (`UnitAnalytics` + вкладки)
- Обзор, расходы по категориям, доходы по категориям, тренды по месяцам.
- Режим: **активный счет** / **все счета**.

### Автосохранение
- Все изменения сразу пишутся в SQLite.
- Запроса «сохранить перед выходом» нет.

---

## Страница «Настройки» (`UnitSettings`)

- **Экспорт CSV** операций активного счета.
- **Импорт CSV** операций (с автоматическим созданием счета, если его нет).
- Создание и удаление счетов; добавление и удаление категорий доходов и расходов.

---

## Диалог записи (`UnitTransactionEditor`)

- Выбор типа: доход/расход.
- Дата: `TDateTimePicker`.
- Категория: выбор из списка или ввод своей.
- Описание и сумма (`> 0`; поддержка точки/запятой в вводе суммы).
- При подтверждении запись сохраняется в SQLite.

## Модули

| Модуль | Содержимое |
|--------|------------|
| `Finance.Types` | Типы операций/счетов, узлы списка, константы. |
| `Finance.Db` | Инициализация SQLite, схема, первичное заполнение. |
| `Finance.Repository` | CRUD для операций/счетов/категорий, настройки, CSV импорт/экспорт. |
| `Finance.Categories` | Обертка совместимости для работы с категориями через репозиторий. |
| `Finance.TransactionList` | Алгоритмы над списком в памяти (суммы, баланс, обход). |
| `Finance.DateUtils` | Разбор даты `дд.мм.гггг`. |
