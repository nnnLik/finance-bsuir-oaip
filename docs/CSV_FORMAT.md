# Формат CSV для импорта/экспорта

## Заголовок

`date_str,is_income,category,description,amount,account_name`

## Поля

- `date_str` — дата в формате `дд.мм.гггг`.
- `is_income` — тип операции: `1` (доход) или `0` (расход).
- `category` — категория.
- `description` — описание.
- `amount` — положительная сумма (`.` или `,` как разделитель дробной части).
- `account_name` — название счета; если такого счета нет, он создается автоматически при импорте.

## Пример

```csv
date_str,is_income,category,description,amount,account_name
06.04.2026,1,Зарплата,Апрель,1500.00,Основной
07.04.2026,0,Продукты,Супермаркет,58.40,Основной
```

## Правила импорта

- Строки с неправильным количеством колонок пропускаются.
- Строки с невалидной суммой или суммой `<= 0` пропускаются.
- Импорт сообщает количество добавленных и пропущенных строк.
